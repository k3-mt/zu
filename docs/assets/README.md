# assets

Media for the docs and the root README.

**Hero demo (`zu-demo.gif`)** — record the offline escalation arc and drop it here:

```bash
# with charmbracelet/vhs, asciinema, or terminalizer — e.g. capturing:
zu demo --offline --type escalation
```

Save the export as `zu-demo.gif`, then uncomment the `<img src="docs/assets/zu-demo.gif" …>`
line in the root [`README.md`](../../README.md) for an animated hero. Keep it lean (≤ ~5 MB,
~760px wide) so the README loads fast.
